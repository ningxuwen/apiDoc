<h3 id="2">2.生成座位信息</h3>

#### 接口名字

* /client/seat/createSeats.json

#### 接口请求方式

* GET/POST

#### 接口请求参数
* lineNo* ：行数
* columnNo* ：列数
* roomId* ：教室id
* seatStates* ：座位状态和pad值（值：座位状态-pad编号,座位状态-pad编号,...）

|60000002|教室id不能为空|

<h3 id="5">5.获取最新版本信息</h3>

#### 接口名字
* /client/version/versionInfo.json

#### 接口请求方式
* GET/POST

#### 接口请求参数
* type* ：客户端类型（1教师端、2学生端）

#### 接口返回值
* 正确返回：
```
{
    "code": 200,
    "description": "请求成功",
    "detail": {
        "code": "1.0.1",
        "createDate": "2017-01-10 14:36:29",
        "id": 1,
        "info": "hhaa",
        "isforced": 1,
        "remark": "",
        "state": 1,
        "type": 1,
        "url": "/a/1.apk"
    }
}
```
* 错误返回:
```
{ "code": xxxxxxx, "description": "xxxxxxxx","detail":null}
```

#### 接口错误码
错误码 | 说明 |
|:-----|:-----
|60000007|客户端类型不合法|
|60000012|版本信息数据异常|

